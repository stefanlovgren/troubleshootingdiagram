Network issue troubleshooting

For problems reaching the server or the server reaching others: confirm ping and DNS, then firewall and security groups, then that the right ports are listening and the service is bound to the expected interface. Use this flow to resolve or escalate.

This sub-chart covers the “Network issues” node from the main troubleshooting chart. See Permissions or Services & config for other problem areas.
