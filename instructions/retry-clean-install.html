How to retry or clean install

Instruction: retry the installer or perform a clean reinstall

What this is

After fixing the underlying issue (disk, dependencies, logs, permissions), you either retry the same installer or remove a partial install and start from a clean state.

How to do it

  1. Retry: Run the installer again with the same options. Ensure no other process is using the install directory or required ports.
  2. Clean install: Uninstall or remove the product (use vendor uninstaller or delete the install directory). Remove any leftover config under /etc or the user home if needed. Then run the installer from scratch.
  3. If the installer has a “repair” or “resume” option, use it only when the docs recommend it; otherwise prefer a full clean install after removing the previous attempt.
