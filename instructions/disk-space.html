How to check disk space

Instruction: verify free space before or after install failures

What this is

Installers often fail or leave the system broken when the target filesystem is full. You need to confirm that the partition(s) used for the installation have enough free space.

How to do it

  1. On Linux, run df -h to see usage of all mounted filesystems. Check the mount point where the server is installed (e.g. /, /opt).
  2. Ensure at least 10–20% free (or the amount recommended by the server docs). If usage is near 100%, free space: remove unneeded files, move logs, or extend the volume.
  3. On Windows, open This PC and check free space on the drive where you install, or run Get-PSDrive -PSProvider FileSystem in PowerShell.
